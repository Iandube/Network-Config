This is a personal project of mine, due to me being in the CTY program of Fanshawe, I am making this project to help speed up networking issues and problems. Such as the coding portion of Cisco switches and routers.

This isn't necessarily a problem, the coding in cisco is well implemented, however it can take quite a bit of time to individually write out the script that your employer or professor could want, even as a personal project.
That is why I am making this, the end goal of this project is to speed up the process by filling in what you need, and then the python script will generate its own cisco script for you, that way you can copy/paste it into the designated networking device.

This is a very early build of the project, it will be updated periodically as I go through my semesters (and I am still a college student). So please be patient with me.
